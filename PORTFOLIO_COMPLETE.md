## 🔧 Customization Quick Guide

### Most Common Changes

#### 1. Update Email

**File**: `src/components/Contact.js` (Line ~65)

```jsx
value: "your-email@example.com",
```

#### 2. Change GitHub URLs

**Files**: `src/components/Header.js`, `src/components/Experience.js`, `src/components/Footer.js`

```jsx
href: "https://github.com/your-username/project-name";
```

#### 3. Update Project Details

**File**: `src/components/Experience.js` (Lines 7-60)

- Project title, description, technologies, links

#### 4. Modify Skills

**File**: `src/components/Skills.js` (Lines 5-42)

- Add/remove skills and adjust proficiency levels

#### 5. Change Colors

**File**: `tailwind.config.js` (Lines 12-20)

```javascript
navy: { 900: "#YOUR_COLOR" }
```
